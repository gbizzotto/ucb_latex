# ucb_latex

*Fork to start a new LaTeX document*

Latex package for academic documents at UCB - Universidade Católica de Brasília.
Contains commands for nicely formatted and spec-conforming:
* front page ;
* equations and results ;
* bibliography.

This is a stub. I am not an experienced LaTeX programmer or even user.

PRs welcome.

Conformity:
* Document as a whole
** Margins (top and left 30mm, bottom and right 20mm)
** Typeface (Times) and font (12pt)
** Text alignment (jusitified)
** Line spacing (1.5)
** Alignment for titles and subtitles (left)
** Text alignment for long titles
** Page number in top right corner
* Front page
** Background image
** Positioning of required elements
** Text color (Blue)
** Text font (18pt)
* Summary
** Title in all caps
* Bibliography
** Title in all caps
** Line spacing (1)
** Text alignment (left)

Perks:
* \eq for slightly larger Equations, for readability
* \eqnum for numbered equations (e.g. (1) a = 0)
* \result for even larger results, centered and boxed
